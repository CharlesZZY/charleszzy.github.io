User Manual for SuperHero Time Table

Introduction

SuperHero Time Table (SuperHeroTT) is an educational Java application designed for children to practice multiplication tables. It features a simple graphical user interface (GUI) with engaging superhero-themed visuals.


System Requirements

  • Java Runtime Environment: OpenJDK 21.0.2 or later
  • Operating System: Compatible with Windows, Mac OS, or Linux

Getting Started

  1. Installation:

    • Ensure Java (OpenJDK 21.0.2) is installed (java -version).

  2. Launching the Application:

    • Open terminal/command prompt in 221170559 directory, then navigate to src/Task1, src/Task2, or src/ExtraCredit.
    • Run the appropriate command:
      • Windows: start build_and_run.bat
      • MacOS/Linux: sh build_and_run.sh

  3. Manual Compilation and Execution:

Using SuperHeroTT

  1. Initial Screen:
    • Upon launching, you will see the main screen of SuperHeroTT as depicted in Figure 1 of the project documentation.
    • Use the drop-down box for Task1 or the 12 checkboxes for Task2 and ExtraCredit to select numbers from 1 to 12.
  2. Starting a Round:
    • Select a number from the drop-down menu or select some numbers from the checkboxes.
    • Click the Start button to begin. The button will change to Next.
  3. Answering Questions:
    • A multiplication question will appear in the Product label in the format x * n, where x is your selected number, and n is a random number between 1 and 12.
    • Type your answer into the answer box and press the Next button or press Enter.
    • The Answer label will show Correct if right or Wrong! x * n = y if wrong, where y is the correct answer.
    • The application will automatically proceed to the next question, and the answer box will be cleared.
  4. Ending a Round:
    • After five questions, the round ends. The Rating label will display You got m correct!, where m is the number of correct responses.
    • The Next button will revert to Start, allowing you to begin another round or select a new number.
  5. Additional Features:
    • Users can change their number selection at any time during the round.
    • If no answer is entered and Next is pressed, the same question will remain without any runtime exceptions.
    • In ExtraCredit, there will be a time limit (countdown) for you.
    • Pressing Enter after input is equivalent to clicking the Next button.

Viewing Javadocs

The SuperHero Time Table project provides detailed Javadoc documentation, which you can view through the following links for each task:

  • Javadocs for Task1: View Task1 Javadocs
  • Javadocs for Task2: View Task2 Javadocs
  • Javadocs for Extra: View Extra Javadocs

These documents include descriptions of all classes, methods, and properties in the project, helping you understand and utilize the SuperHero Time Table application.

You can also view the Javadocs locally:

  • Open the Javadocs:

    • Inside the Documentation folder, you will find a docs directory.

    • Open the docs directory and navigate to the specific task folder (e.g., doc_Task1, doc_Task2, or doc_Extra).

    • Open the index.html file in your web browser to view the Javadocs.

This local copy contains the same documentation as the online version and allows you to access the information even without an internet connection.
